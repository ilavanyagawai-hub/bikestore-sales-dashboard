🚴 BikeStore Sales Dashboard

---

## 📌 About the Project

This project is an end-to-end sales analysis of the BikeStore dataset using Microsoft Excel and Tableau.

The objective was to analyze sales performance, product trends, customer distribution, and regional performance through an interactive dashboard.

The project follows a complete analytics workflow—from data preparation in Excel to building an interactive Tableau dashboard that transforms raw sales data into meaningful business insights.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Identify top-performing product categories.
- Find the highest-selling products.
- Compare sales across different states.
- Analyze monthly sales trends.
- Evaluate store-wise sales performance.
- Identify the best-performing brands.
- Build an interactive dashboard for business insights.

## 📂 Project Workflow

### 1️⃣ Data Preparation (Excel)

- Cleaned and organized the dataset.
- Removed duplicate records.
- Checked for missing values.
- Corrected data types.
- Prepared the data for visualization.

---

### 2️⃣ Dashboard Overview

Built an interactive dashboard featuring:

- Revenue KPI
- Total Units Sold
- Total Customers
- Monthly Sales Trend
- Sales by Category
- Sales by State
- Top 10 Products
- Top 3 Brands
- Store-wise Sales Distribution
- Interactive Filters
The dashboard allows users to interactively explore sales performance using filters and KPIs, making it easier to identify business trends and top-performing products across different dimensions.

## 📈 Key Insights

- Total revenue exceeded **$7.5M**.
- Trek generated the highest revenue among all brands.
- Mountain Bikes contributed the highest sales.
- New York recorded the highest sales among all states.
- April achieved the highest monthly sales.
- Baldwin Bikes generated the highest store revenue.
- A small number of products accounted for a significant share of total sales.
